What is the Document Object Model (DOM)?

Tree Structure: The DOM represents the HTML document as a tree where each node corresponds to a part of the document

Access and Manipulation: Using JavaScript, you can interact with elements in the DOM.

Live Representation: The DOM updates in real-time as the HTML document is modified. For example, if you use JavaScript to change an element, it immediately updates on the webpage

How do you select element for DOM ?

1. Selecting by ID: getElementById() This method selects an element based on its unique id attribute.

Selecting by Class Name: getElementsByClassName() This method returns a live collection (HTMLCollection) of all elements with a specific class name. It can be looped through or indexed.

Live Representation: The DOM updates in real-time as the HTML document is modified. For example, if you use JavaScript to change an element, it immediately updates on the webpage

4. Selecting by CSS Selector: querySelector() This method returns the first element that matches a given CSS selector. The selector can be an ID, class, tag name, or more complex combinations.

5. Selecting by CSS Selector (All Matching Elements): querySelectorAll() This method returns a static NodeList (not live like getElementsByClassName) of all elements that match a given CSS selector.

6. Selecting Form Elements by Name: getElementsByName() This method selects elements based on the name attribute (commonly used in forms).

What is Event delegation in the context of the DOM, and Why is it useful?

1. Event Bubbling: When an event is triggered on an element (like a click on a button), that event doesn't just affect the element itself. It starts at the element and "bubbles up" through its ancestors (the parent, grandparent, etc.) all the way to the document object.

Capturing the Event: Using event delegation, you can listen for the event at a parent level, capture the bubbling event, and determine which child element actually triggered it by inspecting the event's target (event.target).

How Do you manipulate an elements attribute and style using the DOM ?

1. Manipulating Element Attributes To modify or access the attributes of an element (like src, href, id, value, etc.), you can use methods like getAttribute(), setAttribute(), or directly access the properties.

2. Manipulating Element Styles To manipulate the inline styles of an element, you can use the style property of the element. This allows you to set CSS properties directly using JavaScript.

3. Adding or Removing Classes to Manipulate Styles Instead of directly modifying styles, a cleaner approach is to add or remove CSS classes, which allows you to separate logic from presentation.
